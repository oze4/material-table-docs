## Intro 

Component overriding is a big part of Material Table.  Our API has a wide surface area - we expose a lot of our internals which opens the door to some pretty creative solutions.

## Usage 

```jsx
import MaterialTable, { MTableToolbar } from 'material-table';

<MaterialTable
    // other props
    components={{
        Toolbar: props => (
            <div style={{ backgroundColor: '#e8eaf5' }}>
                <MTableToolbar {...props} />
            </div>
        )
    }}
/>
```
