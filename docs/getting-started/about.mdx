## What is `material-table`?

Material Table is a simple and powerful Datatable for React based on [Material-UI](https://material-ui.com/), specifically the [Table](https://material-ui.com/components/tables/#table) component, with some additional features.


:::caution Heads Up
The following guide assumes intermediate level knowledge of HTML, CSS, JavaScript, and React. 
If you are totally new to frontend development, particularly React, it might not be the best idea to jump right into a framework as your first step - grasp the basics then come back! 
:::


## Modular

Material Table is modular in the sense that it exposes a lot of it's internals.  Overriding components allows incredible flexibility - build your perfect table piece by piece!
