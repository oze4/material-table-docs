Actions

You can add buttons to rows or toolbar by using actions prop. Actions prop takes array of actions.

Essentially, Actions are buttons that are located either within the toolbar (to perform table-wide tasks, like export to .csv), or as the first column in each row (to perform row tasks, like edit data in a row).

By default, Actions will be rendered on each row.

Something to remember

Actions are rendered to either the Toolbar, or to the start of each Row

Want to use Actions with Selection?

See here for more info

Usage#

<MaterialTable
  data={data}
  columns={columns}
  actions={[
    {
      icon: 'save',
      tooltip: 'Save User',
      onClick: (event, rowData) => {
        // Perform save action
        console.log({ event, rowData });
      },
    },
  ]}
/>

Demos#

Simple#

Live Editor
function SimpleAction() {
  const data = [
    { id: 0, name: 'Joe' },
    { id: 1, name: 'Mary' },
  ];
  const columns = [
    { field: 'id', title: 'Id' },
    { field: 'name', title: 'Name' },
  ];
  return (
    <MaterialTable
      data={data}
      columns={columns}
      title="Simple Action"
      actions={[
        {
          icon: 'save',
          tooltip: 'Save User',
          onClick: (event, rowData) => {
            alert(JSON.stringify(rowData, null, 2));
            console.log({ event, rowData });
          },
        },
      ]}
    />
  );
}

Caveats#

Using with options.selection#

If you want to use an Action at the start of each Row, as well as use the options.selection, you'll need to specify the location of the Action:

Live Editor
function CaveatOptionsSelection() {
  const rando = (max) => Math.floor(Math.random() * max);
  const words = ['Paper', 'Rock', 'Scissors'];
  const data = [];
  for (let i = 0; i < 100; i++) {
    data.push({
      id: rando(300),
      name: words[i % words.length],
    });
  }
  const columns = [
    { title: 'Id', field: 'id' },
    { title: 'Name', field: 'name' },
  ];
  const actions = [
    {
      name: 'remove', 
      icon: () => <RemoveCircleIcon />,
      tooltip: <h1>I am a tooltip</h1>,
      onClick: (event, rowData) => alert(JSON.stringify(rowData, null, 2)),
      disabled: false, 
      position: 'row',
    },
    {
      name: 'account',
      icon: () => <AccountCircleIcon />,
      tooltip: <h1>I am a tooltip</h1>,
      onClick: (event, rowData) => alert(JSON.stringify(rowData, null, 2)),
      disabled: false,
      position: 'row',
    },
  ];
  return (
    <MaterialTable
      actions={actions}
      data={data}
      columns={columns}
      components={{
        Row: (props) => {
          const propsCopy = { ...props };
          propsCopy.actions.find((a) => a.name === 'remove').disabled =
            propsCopy.data.id < 100;
          propsCopy.actions.find((a) => a.name === 'account').disabled =
            propsCopy.data.name !== 'Paper';
          return <MTableBodyRow {...propsCopy} />;
        },
      }}
      options={{
        selection: true,
      }}
    />
  );
}
