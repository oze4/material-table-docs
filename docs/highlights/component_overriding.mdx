### Important Info

:::note Important Info
 - The `components` property accepts an object [with this shape](#shape-of-components-object)
   - [More info can be found here](#table-of-components-fields)
 - Each field within the `components` object expects a function that returns a `React.Element` type
   - We pass a [props](#link-to-more-about-the-props-object-in-callback) object to said function
:::

## Usage 

```jsx
import MaterialTable, { MTableToolbar } from 'material-table';

<MaterialTable
    // other props
    components={{
        Toolbar: props => (
            <div style={{ backgroundColor: '#e8eaf5' }}>
                <MTableToolbar {...props} />
            </div>
        )
    }}
/>
```
