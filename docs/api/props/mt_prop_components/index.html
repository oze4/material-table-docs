The components Property#

Intro#

  • The components property accepts an object with the following fields/shape
    • More info on each field can be found here
  • Each field within the components object expects a function that returns a React.Element type
    • We pass a props object to said function

Shape#

Optional

These fields are all optional

// See more HERE on the `(props)` object that gets passed
const components = {
  Action: (props) => <React.Element />,
  Actions: (props) => <React.Element />,
  Body: (props) => <React.Element />,
  Cell: (props) => <React.Element />,
  Container: (props) => <React.Element />,
  EditField: (props) => <React.Element />,
  EditRow: (props) => <React.Element />,
  FilterRow: (props) => <React.Element />,
  Groupbar: (props) => <React.Element />,
  Header: (props) => <React.Element />,
  OverlayLoading: (props) => <React.Element />,
  Pagination: (props) => <React.Element />,
  Row: (props) => <React.Element />,
  Toolbar: (props) => <React.Element />,
}
